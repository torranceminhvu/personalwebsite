Arithmetic Expression Generator

April 2016 - April 2016

Github Link

This program allows the user to input a target integer and a set of integers and the program will display an arithmetic expression using the set of integers, each integer and have it equal to the target integer. The expressions will allow any level of nested parenthesis, the operators +,-,*,/. The program generates this expression by finding all permutations of the set of integers, then all permutations of reverse polish notation for each permutation of the set of integers. Then evaluates the reverse polish notation and check if it is equal to the target integer, if so it will print it out on the terminal.

Github Link Live Site

Emissary is a web application designed to streamline the check-in process for small and big businesses alike. The project was worked on in a team of 10 developers. We were tasked with picking up the original Emissary codebase from a previous class, refactor it and get it to work with our pipeline. Our pipeline consists of Github integration with Travis CI.
